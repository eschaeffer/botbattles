What is BotBattles?

BotBattles is an AI‑vs‑AI arena. You build a bot’s brain, then step back and watch what happens.

  • Your bot can look, move, turn, shoot, and decide on its own.
  • Once the match starts, humans are hands‑off.
  • The game is design → test → observe → improve.

There’s no single best strategy—clear, consistent ideas win more than luck.

If this is your first time using BotBattles, the next page describes how to start using the system.

Start with the sample “lame bots,” then improve them.

  1. Download the lameBots.zip file below and unzip/extract it.
    • (Windows) Right click the zip file → Extract All → Extract.
    • (Mac) Double-click the zip file to unzip it.
    • (Linux) Use your file manager’s Extract option or `unzip` in terminal.
  2. Folder layout should look like this:
lameBots/
  Bump/
    bot-bump.json
    behavior.js
  Drift/
    bot-drift.json
    behavior.js
  Idle/
    bot-idle.json
    behavior.js
  ...
  1. Click <Select Bot Folders> and choose the unzipped parent folder (lameBots).
  2. Check the Bot Load Report for errors or warnings.
  3. Click <Start Battle> to begin.

Want better bots? That’s up to <you>.

See the documentation for more detailed information about the BotBattles system.
